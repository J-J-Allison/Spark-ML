# Spark-ML
Projet end-to-end en PySpark sur le dataset Online Retail (UCI Machine Learning Repository) : segmentation client via RFM + clustering (BisectingKMeans, GMM) et classification « gros dépensier » avec Random Forest. Feature engineering avancé, évaluation (F1 0.88) et recommandations business orientées rétention &amp; croissance.
